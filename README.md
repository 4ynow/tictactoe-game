# Tic-Tac-Toe

A command-line Tic‑Tac‑Toe game with an unbeatable AI using the minimax algorithm.

## Features

- Play against the computer (AI with minimax)
- Simple text‑based board display

## How to Run

1. Make sure you have Python 3 installed.
2. Download the script or clone the repository.
3. Run the game:
   ```bash
   python tictactoe.py
